2025/11/26 学习Java前端的第十五天吧（应该是）
今天做完的一件事就是把我之前跟着学习的前端三件套d额一个练习项目自己重新敲了一遍，有一些遗忘，但大部分还是能完成，最终这个小小的管理系统花了大概也有个五小时左右吧。相较于原本课上教的那个代码还是有一些小改变的，也略有一些心得：
1.axios的异步请求还真的注意，我用的是vue3渲染从网站获取的数据，同时又写了一个js小代码来对获取的数据进行一个变色方便查看的处理，但最后写完才发现js代码没有生效，询问ai得到的结论就是js代码和axios请求同时发送，在axios还没有获取到数据时js就已经运行完成了，导致在vue3渲染前js没有找到对应的获取项，最终就是js没起作用。最后的解决办法就是加了一个setTimesout的100ms延时，使js在vue渲染完后再进行处理，成功达成了我想要的效果。至于有没有更好的解决方法的话那肯定是有的，设置延时这事就不太稳定，如果数据太多渲染时间过长那还是会js失效，更好的解决方法应该就是利用css代码来做这个悬停变色的效果，或者说使用一个vue的钩子来进行渲染是否完成的判断，这样的话代码的鲁棒性肯定比我这个高的多，不过我写这个js小功能的原因就是我要进行一个js语法的练习，所以有没有更好的方案，我换不换那还是以后我有闲工夫了再说吧OvO。
